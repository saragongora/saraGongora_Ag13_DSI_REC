# 🎬 Meu Portal de Entretenimento Digital

Este é um projeto simples de página web desenvolvido com HTML e CSS, com o objetivo de criar um portal de entretenimento dividido em três seções principais:

- Filmes e Séries  
- Música e Podcasts  
- Games e Tecnologia  


---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- Flexbox

---

## 🎨 Recursos de CSS

Este projeto utiliza três tipos de CSS:

- **Inline** → aplicado diretamente no elemento HTML  
- **Interno** → dentro da tag `<style>` no `<head>`  
- **Externo** → arquivos `style.css` e `reset.css`

---

## 📌 Funcionalidades

- Layout responsivo com Flexbox  
- Organização em cards por categoria  
- Uso de imagens ilustrativas  
- Estilização com classes e IDs  
- Reset CSS para padronização entre navegadores  

---

## 💡 Objetivo

O objetivo deste projeto é praticar:

- Estruturação de páginas HTML  
- Organização de CSS em diferentes formas  
- Uso de Flexbox para layout  
- Boas práticas de organização de código
